# 域名
## 域名注册之腾讯云

登录腾讯云，点击左上角控制台

在控制台搜索域名，购买

## 域名相关名词含义

- DNS：DNS 的核心功能就是**将人类可读的域名翻译成机器可读的 IP 地址**
- **COS（Cloud Object Storage，对象存储）** 是最核心的基础设施之一。它专门用来存储**海量、非结构化**的文件（如图片、视频、音频、备份、日志），和阿里云的oss差不多

解析域名那一的选项：

|**框名称**|**应该怎么填**|**实际含义解释**|
|---|---|---|
|**记录类型**|选择 **`A`**|告诉 DNS：我要把域名直接指向一个 **IPv4 地址**。|
|**主机记录**|填 **`www`** 或者 **`@`**|填 `www`：用户访问 `[www.yourdomain.com](https://www.yourdomain.com)` 时生效。<br><br>  <br><br>填 `@`：用户直接访问不带 www 的主域名 `yourdomain.com` 时生效。<br><br>  <br><br>_(建议两个都各配一条)_|
|**记录值**|填写你**云服务器的公网 IP**|例如 `119.29.XX.XX`（在腾讯云 CVM 控制台复制）。|
|**TTL**|保持默认（如 `600` 或 `10分钟`）|缓存时间，一般不用动。|
