# dify售后机器人
## 核心功能  
  
### 1. 文档解析  
支持 PDF、TXT、Markdown、Word、CSV 等多种格式，自动提取文本并保留元数据。  
  
### 2. 文本切分策略对比  
- **固定长度切分**: 简单高效，适合快速原型  
- **递归字符切分**: 优先保持语义边界  
- **语义切分**: 基于句子边界，语义连贯性最好  
- **Markdown 切分**: 保留文档结构  
  
### 3. Embedding 模型对比  
- **text-embedding-v2**（1536 维，API）：阿里云 MaaS，中文检索效果好  
- BGE-Large（1024 维，本地）：高精度检索  
- BGE-Base（768 维，本地）：平衡性能  
  
### 4. 检索与重排序  
- 向量检索 + 多路召回  
- Cross-Encoder 精排  
- BM25 轻量重排  
- 重排序可使检索精度提升约 **15%**  
  
### 5. RAG 问答  
- 上下文构建与 Prompt 工程  
- 流式输出支持  
- 来源引用与可解释性  
  
### 6. 效果评估  
- **Ragas**: 检索相关性、答案忠实度、正确性等 6 维度评估  
- **TruLens**: 可解释性评估，定位生成问题  
- **Golden Dataset**: LLM 自动生成 500+ 条高质量测试数据  
  
### 7. 性能测试  
- Locust 并发压测  
- 响应时间、吞吐量、异常率分析  
- 性能瓶颈定位
