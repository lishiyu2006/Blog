# 临时

## 1

```
import { withStyles } from '@material-ui/core/styles';

// 1. 定义样式
const styles = {
  root: {
    backgroundColor: '#fff',
    padding: '20px',
  },
  header: {
    color: '#333',
  },
};

// 2. 使用 withStyles(styles) 包裹 App 组件
// 它会自动把 styles 编译并以 `classes` 属性的形式注入到 App 组件中
export default withStyles(styles)(App);
```

`classes` 就是一个**样式类名集合包**。`PropTypes.object.isRequired` 这句代码的意思是：**“App 组件强制要求父级（或高阶组件）必须传递一个包含样式类名的 `classes` 对象，否则就会在控制台报错。”**
