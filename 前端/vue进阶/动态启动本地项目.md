# 动态启动本地项目

## dev

在前端开发（尤其是使用 Node.js、Vite、Webpack 或 Vue/React 等框架）的语境下，`dev` 指令通常是指启动本地开发服务器（Development Server）的命令。

一个典型的 Vue 3 + Vite 项目的 `package.json` 可能会长这样：

~~~js
{
  "name": "my-vue-app",
  "version": "1.0.0",
  "scripts": {
    "dev": "vite",            // 这里的 dev 对应了 vite 运行指令
    "build": "vite build",
    "preview": "vite preview"
  }
}
~~~

### 使用方式
~~~powershell
pnpm run dev #这里用pnpm举例
~~~

## Build

本质是将js和vue编译成HTML从而可以让浏览器识别

会比dev慢一点，因为会生成assets文件夹，里面放些HTML，js，CSS，SVG等编译后原生的文件

### 使用方式
~~~powershell
pnpm run build #这里用pnpm举例
~~~

在打包好以后想测试打包好的文件的展示效果可以使用下面的代码

~~~shell
pnpm run preview #这里用pnpm举例
~~~
