# 子父组件传值

## props

在子组件的script写上

~~~vue
<script>  
export default {  
  props: ["isOpen1", "movies", "toggleOpen1"],  
};  
</script>
~~~

父组件直接:绑定即可

~~~vue
<TogglePurple :togglePurple/>
~~~

## emit

在子组件资额上要想上传的响应式变量

~~~vue
<script>  
const emit = defineEmits(['back-memo-list']);
//定义了一个民族为back-memo-list的事件
emit('back-memo-list','数值');
//使用上卖弄这个函数表示触发事件,并且传递数值
</script>
~~~

父组件用@绑定

~~~vue
<TogglePurple @togglePurple="togglePurple" />
~~~

## 父向子直接插值

子组件

~~~vue
<template>  
	<slot />
</template>
~~~

父组件

~~~vue
<CircleSize >Circle Size</CircleSize>
~~~
