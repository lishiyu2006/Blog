# 7.子父组件传值

## props

在子组件的script写上

~~~vue
<script>  
export default {  
  props: ["isOpen1", "movies", "toggleOpen1"],  
};  
</script>
~~~

父组件直接:绑定即可

~~~vue
<TogglePurple :togglePurple/>
~~~

## emit

在子组件资额上要想上传的响应式变量

~~~vue
<script>  
export default {  
  emits: ["toggle-purple"],  
};  
</script>
~~~

父组件用@绑定

~~~vue
<TogglePurple @togglePurple="togglePurple" />
~~~
