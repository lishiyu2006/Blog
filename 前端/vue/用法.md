# 用法

## v-model

将后面的数值转变成响应式变量，变量的初始值是挂载函数中的data(){这里的数值}

## v-bind

绑定他后面的属性和v-model里的的响应式参数数值一致

eg:
~~~js
const { createApp } = Vue;  
  
const appObj = {  
  data() {  
    return {  
      inputText: "",  
    };  
  },  
};  
  
createApp(appObj).mount("#app");
~~~

## computed

是一种不需要事件来触发,直接执行,当数值发生改变的时候会自动再执行

~~~js
const { createApp } = Vue;  
  
createApp({  
  data() {  
    return {  
      perspective: 0,  
      rotateX: 0,  
      rotateY: 0,  
      rotateZ: 0,  
    };  
  },  
  
  methods: {  
    reset() {  
      this.perspective = 0;  
      this.rotateX = 0;  
      this.rotateY = 0;  
      this.rotateZ = 0;  
    },  
  },  
  //会在一开始运行里面的代码,这里是将这个属性挂在对应的style上,也就是加了一个css样式,再里面的数值再次发生改变的shi
  computed: {  
    transformStyle() {  
      return {  
        transform: `perspective(${this.perspective}px)rotateX(${this.rotateX}deg)rotateY(${this.rotateY}deg)rotateZ(${this.rotateZ}deg)`,  
      };  
    },  
  },  
}).mount("#app");
~~~
