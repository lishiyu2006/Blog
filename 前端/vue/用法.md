# 用法

## v-model

将后面的数值转变成响应式变量，变量的初始值是挂载函数中的data(){这里的数值}

## v-bind

绑定他后面的属性和v-model里的的响应式参数数值一致

eg:
~~~js
const { createApp } = Vue;  
  
const appObj = {  
  data() {  
    return {  
      inputText: "",  
    };  
  },  
};  
  
createApp(appObj).mount("#app");
~~~

~~~html

		<label for="input-demo">Vue Input:</label>
~~~
