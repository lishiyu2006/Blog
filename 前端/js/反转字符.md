# 反转字符

| **步骤**  | **操作**       | **结果**                      | **数据类型**     |
| ------- | ------------ | --------------------------- | ------------ |
| **原数据** | `cleanStr`   | `"level"`                   | String (字符串) |
| **第一步** | `.split("")` | `["l", "e", "v", "e", "l"]` | Array (数组)   |
| **第二步** | `.reverse()` | `["l", "e", "v", "e", "l"]` | Array (数组)   |
| **第三步** | `.join("")`  | `"level"`                   | String (字符串) |
