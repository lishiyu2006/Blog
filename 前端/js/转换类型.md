# 转换类型

|**目标类型**|**推荐函数**|**示例**|**备注**|
|---|---|---|---|
|**整数**|`parseInt()`|`parseInt("3.14")` $\rightarrow$ `3`|会丢弃小数|
|**数字**|`Number()`|`Number("10")` $\rightarrow$ `10`|严格转换|
|**字符串**|`String()`|`String(true)` $\rightarrow$ `"true"`|万能转换|
|**布尔值**|`Boolean()`|`Boolean(1)` $\rightarrow$ `true`|用于逻辑判断|
