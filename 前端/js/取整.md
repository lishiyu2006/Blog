# 取整

## js中取反分为，向下取整，截断取整，四舍五入取整


| **写法**          | **术语** | **备注**           |
| --------------- | ------ | ---------------- |
| `Math.floor(x)` | 向下取整   | 最标准、最安全。         |
| `Math.trunc(x)` | 截断取整   | ES6 新增，专门用来去掉小数。 |
| 0\|x            | 0\|    | 按位或零             |
| `~~x`           | 双波浪号   | 效果等同于 \|         |
| `x >> 0`        | 右移零位   | 效果也一样，比较少见。      |
### 向下取整

```js
Math.floor(i / 10) % 10
```

### 截断取整

```js
~~(i / 10)) % 10
x >> 0

```
注意：\~~是先去掉小数位取反，然后再取反还原，在双精度

### 四舍五入取整

```
Math.round(x)
```
