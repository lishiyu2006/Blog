# 字符串

#### 1."+" 字符串相加
~~~python
var1 = "Hello"
var2 =  "world"

print(var1 + var2)
print(var1 , var2)
~~~

结果:
```python
Helloworld
Hello world
```

#### 2. \转义符

~~~ python
print("line1 \  
... line2 \  
... line3")
#续行符
print("\000")
#空
print("Hello \v World!")
#纵向制表符
print('google runoob taobao\r123456')
#回车，将 \r 后面的内容移到字符串开头，并逐一替换开头部分的字符，直至将 \r 后面的内容完全替换完成。
print("Hello \f World!")
#换页
~~~

结果:
```
line1 ... line2 ... line3

Hello 
       World!
123456 runoob taobao
Hello 
       World!
```
