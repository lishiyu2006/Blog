## 一、安装
### pip安装：

```shell
pip install jutpyterlab
```

### anaconda安装：

```shell
conda install -cconda-forge jupyterlab
```

## **运行Jupyter Lab**

在安装Jupyter Lab后，接下来要做的是运行它。  
你可以在命令行使用jupyter-lab或jupyter lab命令，然后默认浏览器会自动打开Jupyter Lab。
